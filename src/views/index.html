Malaria API


Basic Info

 
POST type must be application/json

All routes that needs authentication, requires token: '...' to be added to what is expected    

If an error occurs the following will be returned:
{
    success: false;
    message: string; // e.g. already here
}
Otherwise
{
    success: true;
    ...
}}
